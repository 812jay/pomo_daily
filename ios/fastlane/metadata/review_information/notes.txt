[앱 소개]
Pomo Daily는 뽀모도로 기법을 활용한 시간 관리 앱입니다.
사용자가 원하는 시간으로 작업/휴식 시간을 설정할 수 있습니다.

[주요 기능]
1. 타이머 기능
- 작업 시간 설정 
- 휴식 시간 설정
- 세트 수 설정 가능
- 자동 재생 기능
- 진동 알림 기능
- 언어 변경 기능
- 다크모드 기능

2. 설정 기능
- 진동 ON/OFF
- 언어 설정 (한국어/영어)
- 다크모드 기능

[테스트 방법]
1.타이머 기본 테스트
- 홈 화면에서 시작 버튼을 눌러 타이머 시작
- 일시정지/재시작 가능
- 세트 완료 시 자동으로 다음 모드(작업/휴식)로 전환
- 타이머 완료 시 진동 알림 확인
2.타이머 설정 변경 테스트
- 타이머 동작 중 설정 화면으로 이동
- 집중 시간/휴식 시간/세트 수 변경
- 저장 버튼 클릭 시 확인 다이얼로그 출현 확인
- "저장" 선택 시 타이머 초기화 및 새로운 설정 적용 확인
- "취소" 선택 시 이전 설정 유지 확인
3.자동 재생 테스트
- 타이머 설정에서 자동 재생 ON
- 세트 완료 시 자동으로 다음 세션 시작
- 전체 세트 완료 시 축하 화면 확인
4.설정 테스트
- 설정 화면에서 진동/언어 설정 변경 가능
- 타이머 설정에서 시간 조정 가능
- 설정 화면에서 다크모드 조정 가능
